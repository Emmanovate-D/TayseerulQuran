# TayseerulQuran - Learning Management System

A comprehensive Learning Management System (LMS) for Quranic education.

## Project Structure

- **backend/** - Node.js/Express backend API
- **TayseerulQuran/** - Frontend HTML/CSS/JavaScript files

## Features

- User authentication and authorization
- Course management
- Student and tutor management
- Payment processing
- Blog system
- Role-based access control (RBAC)

## Setup

See the documentation in the `backend/` folder for setup instructions.

## Repository

This repository contains the complete source code for the TayseerulQuran LMS project.
